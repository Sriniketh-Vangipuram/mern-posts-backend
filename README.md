# MERN Posts Backend

Express REST API for the MERN Posts application.

Frontend Repository

https://github.com/Sriniketh-Vangipuram/mern-posts-frontend/tree/main

Frontend Live URL

https://mern-posts-frontend-seven.vercel.app/

Backend API

https://mern-posts-backend.onrender.com

---

## Features

* REST API
* MongoDB Atlas integration
* Image uploads using Multer
* Cloudinary storage
* Persistent database
* CRUD operations
* CORS configuration

---

## Tech Stack

* Node.js
* Express
* MongoDB Atlas
* Mongoose
* Multer
* Cloudinary
